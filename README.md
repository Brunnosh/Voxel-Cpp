## Build (WINDOWS)

### *VSCode*
Install:
- Cmake
- GCC (MSYS2-UCRT64): `pacman -S mingw-w64-ucrt-x86_64-gcc`
- Ninja (MSYS2-MINGW): `pacman -S mingw-w64-x86_64-ninja`

**(Remember to add them to PATH) (ucrt64/bin and mingw64/bin)**

Inside VsCode:
- CTRL+SHIFT+P : `CMake: Select Configure Preset` (choose debug/release)
- .vscode folder: 
    - c_cpp_properties : Change C compiler path and include folders

CTRL+SHIFT+B builds project, latest executable (Debug or Release) will be found on `root/latest`, press F5 to launch from VsCode


### *VisualStudio*

if having some issues, maybe change C compiler from MSVC to GCC, but should  just be open & build.
